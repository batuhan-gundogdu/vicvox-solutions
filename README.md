# VicVox Solutions Dashboard

Live, aggregate-only metrics from the [VicVox](https://www.vicvox.ai) civic
discourse pipeline, auto-published after every crawl.

**Dashboard: https://batuhan-gundogdu.github.io/vicvox-solutions/**

No individual-level data is published here: the pipeline's privacy gate
structurally rejects usernames, comment text, identifiers, and inferred
demographics before anything reaches this repository. `data.json` contains
only counts, dates, regions, and community-level metrics.
